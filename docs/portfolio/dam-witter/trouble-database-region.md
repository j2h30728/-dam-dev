# 데이터베이스 리전

- **문제**: 프로젝트 초기 단계에서 **API 응답 시간이 지나치게 느려**서 성능 저하 문제를 겪었습니다. 이 문제는 **데이터베이스 리전**이 미국으로 설정되어 있었기 때문에 발생했습니다.
- **해결**: 데이터베이스 리전을 **일본으로 변경**하여, 사용자와 가까운 서버로 이동시켜 응답 시간을 대폭 개선했습니다.
